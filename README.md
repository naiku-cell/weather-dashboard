Weather Dashboard

Description

An interactive weather dashboard that allows users to search any city and view current weather conditions.  
Uses Open-Meteo API to fetch weather data and displays human-friendly descriptions.


Features
- Search any city for current weather
- Interactive UI with temperature, wind, and weather description
- Human-friendly weather codes and icons
- Responsive layout for desktop and mobile
- Hosted on GitHub Pages


Technologies Used
- HTML
- CSS
- JavaScript
- Open-Meteo API


 How to Run
     
Open the live demo: [Weather Dashboard](https://naiku-cell.github.io/weather-dashboard/)
